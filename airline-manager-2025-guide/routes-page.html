Enter the Command

Go to $bot-spam or $fun-bot-spam.
Enter the following command in this format: $routes [airport code] [airplane code]
Example:
$routes ORD B738 <- Code for Chicago's O'hare International Airport and Boeing 737-800


Download the Root Information

Below the message, you will see an option called CSV. Select it and download the data provided. From there, upload the data/table to Google Sheets or Microsoft Excel.


Understanding the Root Information

Destination Name: Where the flight is going
Destination Country: The country the flihgt is going
Stopover Name: The airport where the plane will stop
Stopover Country: The country the stopover airport is located
Seat Configurations: Economy (Y), Business (J), First (F)



Configure Your Planes

Example from the video:
1. Economy Seats: 322 seats
2. Business Seats: 139 seats
3. First Seats: 0 seats
Using the data provided from the table, enter the numbers into the game. This is great way to maximize seat usage while maximizing profits in your aircraft.


Set Up Your Route

Enter the stopover airport then enter the destination's airport.
Set the ticket prices for each class.
Now, you are all set to depart your aircraft.
